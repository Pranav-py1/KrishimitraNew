# **App Name**: KrishiMitra

## Core Features:

- Farmer Authentication: Implement phone OTP login for farmers using Firebase Authentication.
- Browse Machines: Allow farmers to browse available machines for rent.
- Browse Agri Products: Enable farmers to browse available seeds and other agricultural products.
- Machine Booking: Allow farmers to book machines for rent.
- Soil Test Booking: Enable farmers to book offline soil testing services.
- Booking History: Allow users to view booking history and status.
- Retailer Registration: Allow retailers to register their shop with necessary details.

## Style Guidelines:

- Primary color: Rich Green (#2E7D32) to represent growth, agriculture, and trust, creating a natural and reliable atmosphere.
- Background color: Light Soft Green (#E8F5E9) to create a fresh, clean, and comfortable interface that is easy on the eyes for rural users.
- Accent color: Warm Yellow (#F9A825) for interactive elements and calls to action, giving high visibility and energetic contrast against the primary colors.
- Typography: Use ‘PT Sans’, a humanist sans-serif font chosen for clarity, readability, and accessibility across both mobile and desktop screens.
- Iconography: Use simple, easily recognizable icons to ensure clear understanding for users with varying levels of digital literacy.
- Layout: Implement a grid-based layout for consistent spacing and alignment, ensuring intuitive navigation and a clean user experience.
- Animation: Use subtle transitions to provide feedback and guide users through interaction, enhancing the overall usability without distraction.
